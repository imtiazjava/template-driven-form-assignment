# template-driven-form-assignment

Devlop the template driven form for the registeration by using the following fields and constraints:
--------------------------------------------------------------------------------------------------

FIELDS:

- FIRST NAME
- LAST NAME
- EMAIL ID
- PASSWORD
- RESET PASSWORD
- MOBILE NO
- GENDER 
- ADDRESS 

RULES:

FIRST NAME 
- SHOULD BE MINIMUM 15 CHARACTERS
- SHOULD NOT BE BLANK

LAST NAME
 -SHOULD BE MINIMUM 15 CHARACTERS
 - SHOULD NOT BE BLANK
 
 EMAIL ID
 - SHOULD NOT BE BLANK
 - SHOULD BE VALID
 
 PASSWORD
 - SHOULD NOT BE BLANK
 - SHOULD CONTAIN COMBINATION OF NUMBERS , CHARACTERS
 
 MOBILE NO
 - SHOULD NOT BLANK
 - SHOULD CONTAIN ONLY 10 DIGITS
 
 GENDER , ADDRESS
 - SHOULD NOT BE BLANK
